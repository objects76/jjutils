# jjutils

유틸리티 함수 모음 패키지

## 설치 방법

```bash
# pip로 GitHub 저장소에서 직접 설치
pip install git+https://github.com/objects76/jjutils.git

# 또는 로컬 복제 후 설치
git clone https://github.com/objects76/jjutils.git
cd jjutils
pip install -e .
```
